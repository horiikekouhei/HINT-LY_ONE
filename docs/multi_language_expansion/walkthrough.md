# 多言語対応の拡張（合計9言語対応）

既存の日本語・英語に加え、世界で話者数が多い主要言語と指定の言語（計7言語）を追加し、合計9言語でプレイ可能になりました。

## 追加された言語
- **中国語（簡体字）** - zh-CN
- **台湾語（繁体字）** - zh-TW
- **韓国語** - ko
- **スペイン語** - es
- **ヒンディー語** - hi
- **アラビア語** - ar （右から左への記述方向にも対応）
- **フランス語** - fr

## 実装の詳細

### 1. 翻訳辞書の拡充
- `src/i18n/locales/` 配下に各言語の翻訳ファイルを新規作成しました。UI上のすべてのテキスト（ルール説明、各フェーズのガイド、エラーメッセージ等）がそれぞれの言語で表示されます。

### 2. お題リストの多言語化
- `src/data/topics.ts` に各言語専用の単語リスト（各約150単語）を追加しました。
- 部屋の作成時に選択されている言語が「部屋の言語」として固定され、ゲーム中はその言語のお題が出題されます。

### 3. UIの改善
- 言語数が増えたため、ロビー画面右上の切り替えUIを従来のトグルボタンからドロップダウンメニュー（`<select>`）に変更しました。
- アラビア語選択時には、ブラウザの表示方向が自動的に右から左（RTL）に切り替わるように調整しました。

### 4. ロジックの汎用化
- `gameStore.ts` において、言語コードから適切なお題リストを動的に取得するヘルパー関数を導入し、今後さらに言語を追加する場合も容易に拡張できる設計にしました。

## 動作確認の手順
1. ロビー画面右上のドロップダウンから各言語を選択し、UIが正しく切り替わるか確認してください。
2. 特にアラビア語（العربية）を選択した際、レイアウトが反転して右から左の表示になることを確認してください。
3. 任意の言語で部屋を作成し、ゲームを開始した際に出題されるお題（Topic）がその言語になっているか確認してください。
