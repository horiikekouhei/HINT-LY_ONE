# 「Just One」機能拡張の完了

以下の新機能および仕様変更を全て実装しました。

## 実装された機能

### 1. ヒントの書き直し
Phase 2（ヒント入力）において、送信後結構でも「書き直す」ボタンを押すことで、他のプレイヤーが全員入力し終える前であればヒントを修正できるようになりました。

### 2. プレイヤー追放（キック）機能
ホスト（部屋の作成者）は、待機中およびゲーム中にプレイヤーを追放できるようになりました。
- 待機画面のプレイヤーリスト横の「✕」ボタン
- ゲーム画面フッターのプレイヤーチップ横の「✕」ボタン
追放されたプレイヤーは自動的にロビー画面に戻されます。

### 3. スコアリングの変更
- **正解**: +1点
- **不正解**: -1点
- **パス**: 0点
※スコア計算の緊張感が増しました。

### 4. 終了画面の履歴表示
ゲーム終了時のサマリー画面で、各ラウンドの詳細（お題、回答、全員が出したヒントと重複の有無）をリスト形式で確認できるようになりました。

### 5. ラウンド数選択とフリーモード
ロビーでの部屋作成時に以下の設定が可能になりました：
- **ラウンド数**: 5, 13, 20 ラウンドから選択
- **フリーモード**: 無限ラウンド。途中参加が可能で、ホストが好きなタイミングで終了ボタンを押して結果画面へ進むことができます。
