# 多言語表示機能および部屋の言語設定対応

ゲームのUIの多言語化（日本語/英語の切り替え）と、部屋作成時に指定した言語でゲームが進行する機能を実装しました。

## 実装内容

### 1. 多言語切り替え機能 (i18n)
- 外部ライブラリに依存せず、軽量な `LanguageContext` (`src/i18n/LanguageContext.tsx`) を作成しました。
- ユーザーが選択した言語（`ja` または `en`）は `localStorage` に保存され、次回アクセス時にも維持されます。
- `Lobby.tsx` の右上に「🌐 JP / EN」の切り替えボタンを設置し、クリックで瞬時にUI全体の言語が切り替わるようにしました。
- 全ての画面 (`Lobby`, `WaitingRoom`, `Game`, および各Phaseコンポーネント) に翻訳関数 `t()` を適用しました。

### 2. 言語ごとの翻訳ファイルの作成
- `src/i18n/locales/ja.ts` と `en.ts` を作成し、ゲーム内で使用されるすべてのテキストをマッピングしました。

### 3. 部屋ごとの言語設定
- 部屋の型定義 (`Room`) に `language` プロパティを追加しました。
- ロビーで部屋を作成する際、その時点のユーザーのUI表示言語が「部屋の言語」としてFirebaseに保存されます。
- ロビーの「新しく部屋を作る」画面で、「部屋の言語: 日本語/English」と明記されるようにしました。

### 4. 言語ごとのお題リスト
- `src/data/topics.ts` を修正し、これまでの `TOPIC_LIST` を `TOPIC_LIST_JA` とし、同等のカテゴリを持つ英単語のリスト `TOPIC_LIST_EN` を新規追加しました。
- ゲーム開始時（または次のラウンドへ進む時）、`room.language` の値に応じて、日本語のお題リストか英語のお題リストかが選ばれるようにしました。

## 動作確認の手順

1. 画面右上の `EN/JP` ボタンでUI言語が切り替わることを確認してください。
2. `EN` (英語) を選択した状態で部屋を作成し、ゲームを開始すると、出題されるお題が英語 (`TOPIC_LIST_EN`) から選ばれることを確認してください。
3. 他の参加者が `JP` (日本語) の状態でその部屋に入った場合、参加者のUI（ボタン等）は日本語のままですが、お題のみ英語で出題される仕様になっています。
