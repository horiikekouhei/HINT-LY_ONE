# お題リストの1000単語への拡張プラン

現在約300語程度（日本語は約1000語）のお題リストを、全言語で1000単語以上に拡張します。

## 実施内容

1.  **カテゴリーの拡充**:
    単語数を増やすため、既存のカテゴリー（食べ物、動物など）を深掘りするとともに、新しいカテゴリー（世界の国、歴史的キーワード、最新のテクノロジー用語など）を追加します。
2.  **全言語への反映**:
    英語をベースに1000語のリストを作成し、それを各言語（中国語、台湾語、韓国語、スペイン語、フランス語、ヒンディー語、アラビア語）に翻訳したリストを作成します。
3.  **ファイルサイズへの配慮**:
    合計で約9000単語（9言語 × 1000語）となるため、`src/data/topics.ts` のファイルサイズが大きくなります（約200KB〜300KB程度を想定）。これはWebアプリとしては許容範囲内ですが、管理を容易にするため、カテゴリーごとにコメントで区切って整理します。

## 作業の進め方

データ量が非常に多いため、数回に分けてファイルを更新します。
- **ステップ1**: 英語・中国語（簡体字）・台湾語（繁体字）の拡張
- **ステップ2**: 韓国語・スペイン語・フランス語の拡張
- **ステップ3**: ヒンディー語・アラビア語の拡張

## ユーザー確認事項

> [!IMPORTANT]
> 1000単語という大量のデータを生成するため、多少お時間がかかる場合があります。また、単語の重複チェックも行いますが、万が一重複があった場合はご容赦ください。
